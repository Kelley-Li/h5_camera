# h5_camera

一个高仿原生安卓的h5照相机

[线上地址](http://auven.github.io/h5_camera/)

## 功能

- 支持PC端和移动端（uc浏览器暂不支持，之后我再试试其他办法）
- 移动端支持切换前后置摄像头
- 全屏展示
